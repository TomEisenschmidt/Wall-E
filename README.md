# Projet IOT MAKER

Création d'un Wall E

## Fonctionnalitées : 

Wall E peut :
- Se déplacer
- Ouvrir et fermer sa trappe
- Déplacer ses bras d'avant en arrière
- Ecran affichant la batterie

## Matériel : 

- 7 moteurs répartis sur différentes tâches :
	- déplacement du robot
	- déplacement des bras
	- ouverture / fermeture de la trappe
- batteries

- différents capteurs : 
	- capteur ToF (Time-of-Flight) : ultrason / infrarouge / laser ?
